# IEXCloudDotNet
IEXCloudDotNet is a .Net class library allowing easy access to the IEXCloud API

# Target Release Date for Version 1.0
- March 15th 2019
# Version 1.0

## Features 
##### Classes Implemented
- BalanceSheet
- Company
- DelayedQuote
- Dividend
- Earnings
- HistoricalPrice
- KeyStats
- Logo
- MarketVolumeUS
- News
- OHLC (Open High Low Close)
- PreviousDayPrice
- PriceTarget
- Quote
- SectorPerformance
- VenueVolumn

All classes implemet INotifyPropertyChange and are set up to work with binding.
